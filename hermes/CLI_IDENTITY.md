# HERMES-PRIME — Custom CLI Art & Terminal Animation System

## Design Philosophy

HERMES-PRIME should not feel like a generic terminal utility.
It should feel like:

* A sovereign cognitive infrastructure
* A classified systems console
* A deterministic governance engine
* A high-trust operator environment
* A living orchestration core

The CLI identity should communicate:

* Precision
* Authority
* Signal dominance
* Controlled intelligence
* Cold operational clarity

No meme-tier hacker aesthetics.
No random matrix rain spam.
No overloaded neon chaos.

The visual language should resemble:

* Aerospace systems
* Autonomous command infrastructure
* Naval combat systems
* AI governance architecture
* Deep-space telemetry consoles

# ELITE TERMINAL EXPERIENCE RULES

## DO

* Keep spacing symmetrical
* Prefer clean geometry
* Use silence and empty space strategically
* Animate slowly and intentionally
* Make the terminal feel expensive
* Keep authority-focused typography
* Maintain deterministic aesthetics

## DO NOT

* Spam random animations
* Overuse RGB effects
* Create fake hacker visuals
* Use meme typography
* Use massive walls of blinking text
* Make it look like a movie prop
* Turn the CLI into visual noise
